# Serenity 卡点分析 — 参考

## 五因子卡点模型

对每个 **产业链节点** 评估（不是对股票）：

| 因子 | 通过标准 | 常见反例 |
|------|----------|----------|
| 1 确定需求 | Megatrend 有 hyperscaler/CAPEX/订单/政策等硬证据 | 只有媒体热度 |
| 2 供给约束 | 产能、认证、工艺、资源、设备交付周期形成瓶颈 | 充分竞争、 commoditized |
| 3 低市场关注 | 非 Mag7、非 everyone 的 pick | 已是 consensus trade |
| 4 价值捕获 | 该环节能涨价、锁单、提份额 | 价值被下游/integration 吃掉 |
| 5 近端催化 | 3–12 月内可验证的里程碑 | 纯 3–5 年故事 |

**锁定规则**：五因子中至少 4 项为「通过」，且因子 1+2 必须通过。

## 市场标签陷阱（卡点归属常见错误）

**反例（真实发生）**：

> 用户问"内存缺货 → A 股卡点"。AI 给出的报告把 **华天科技 002185** 列为「HBM 卡点 A 直接受益」，理由是"HBM 封测三强"。
>
> 事实核查：华天科技 2024/12/29 在深交所互动平台**明确**回复投资者：
> > "公司**尚未开展 HBM 存储器封装业务**，HBM4 对公司现有业务无重大影响。"
>
> 实际卡点（**现在**）：DDR5 通用封装（国内市占率 >40%）+ 长江存储最大外部封测商（40-45% 外包份额）—— 应该归到「利基 DRAM 封测」和「国产 NAND 封测」卡点，不是 HBM 卡点。
>
> 未来卡点（2027+）：TSV 已量产、2.5D/3D 小批量、储备 HBM 国产替代能力 —— 但**不是现在卡点**。

**陷阱模式**：

| 市场标签 | 实际可能 |
|----------|----------|
| "HBM 封测三强" | 三家可能都没在做 HBM，只做通用封测 |
| "存储封测龙头" | DDR4/DDR5 通用封测 ≠ HBM 封测 |
| "国产替代核心" | 储备能力 ≠ 量产收入 |
| "国家队" / "战略合作伙伴" | 客户认证 ≠ 实际采购 |
| "ChatGPT 供应商" | 媒体转述 ≠ 公司确认 |
| 行业研报"前 N 大" | 排名口径经常混算 |

**正确做法**（**Step 5 硬性要求**）：

1. **核公司自己说过什么**：
   - 优先看「公司互动平台」官方回复（深交所互动易 / 上证 e 互动）
   - 搜索格式：`公司名 + "互动平台" / "投资者关系" / "尚未开展" / "暂未涉及"`
2. **核主营业务收入构成**：
   - 季报 / 年报的「主营业务分行业 / 分产品」表，看清具体收入来源
3. **区分现在 vs 未来**：
   - 现在：实际产生收入、有客户、有订单
   - 未来：研发布局、产能建设中、储备阶段、技术 demo
   - 只有「现在」能进 Step 5 受益标的；「未来」放 Step 7 反证或观察信号
4. **找不到公司自己确认就降级**：
   - 找不到 → 标"卡点成立但公司未明确参与"
   - 不要硬凑"市场普遍认为"作为踩点依据

## 业绩兑现陷阱（卡点归属 vs 业绩兑现的鸿沟）

**这是「四件套」防不住的更隐蔽陷阱**——公司确实在该卡点有业务（卡点归属 ✅），但**业绩没兑现**（卡点弹性 ❌）。

**反例（真实发生）**：

> 用户问"光纤涨价 → A 股上游卖水人"。AI 给出报告把 **石英股份 603688** 列为「⭐⭐ 上游卖水人，光棒扩产直接拉动」，理由是"全球三强 + 光纤级市占 80% + 光棒扩产直接拉动"。
>
> 事实核查（按 6 件套核完）：
> 1. ✅ 公司互动平台 2026/4/27："面向光纤行业提供高纯延长管、把手棒、炉芯管等"
> 2. ✅ 招股书+年报：主营高纯石英砂+石英管，光纤业务在「光纤半导体」板块
> 3. ✅ 季报主营分产品：**2025 光纤半导体收入 5.16 亿（+3.24%），占 51.22%**——**确实有这业务**
> 4. ✅ 公告：10 万吨高纯砂产能、半导体三期 6 万吨投产
> 5. **❌ 业绩兑现验证失败**：
>    - 2025 整体业绩：**营收 10.08 亿 -16.71%，归母净利 1.53 亿 -54.03%**
>    - 2025 光纤业务增速 **+3.24%** ≪ 长飞 +226%、亨通 +98.5%
>    - **业绩 -54% + PE(TTM) 274x** = **严重透支**
> 6. ❌ 估值合理性：**业绩 -54% + PE 274x → 必须警示**
>
> 实际卡点：石英股份做的是**光棒辅材**（衬管/套管/炉芯管），不是光棒本身；主业是**高纯砂供半导体+光伏**，光纤只是辅材；2025 光纤业务 +3.24% 实际**没吃到卡点红利**。
>
> 正确标的（按 6 件套重核）：**菲利华 300395**
> - 公司互动平台："与主要光纤光棒厂家保持长期战略合作"
> - 2025 营收 20.16 亿 +15.76%，归母净利 4.43 亿 +41.04%
> - 2026Q1 营收 +53.04%，归母净利 +36.77%
> - PE(TTM) 108.82x 合理
> - 多极增长：半导体 + 军工 + Q 布 + AI 算力 + 光通信

**陷阱的根因**：

| 层次 | 防什么 | skill 是否防了 |
|------|--------|----------------|
| **信源层**（第 1-4 件） | 公司是否在该卡点有业务 | ✅ 防了 |
| **业绩层**（第 5-6 件） | 业绩是否真兑现 | ❌ 之前没防 |

**业绩兑现的具体降级规则**：

| 业绩数据 | 降级动作 |
|----------|----------|
| 该业务在主营分产品表里找不到 | 降级（无业务） |
| 占比 < 5% | 降级（盘子小） |
| 该业务增速 < 公司整体平均增速 | **降级**（没吃到卡点红利） |
| 公司整体 -10% 以上 + 该业务仍 < 行业 | **降级**（无 alpha） |
| 公司整体 -10% + PE > 50x | **警示**（严重透支） |
| 业绩增速 < PE 的 1/2 | **警示**（赔率不足） |

**「业绩兑现验证」的标准检查步骤**：

1. 查最新一期年报/季报的「主营业务分行业/分产品」表
2. 找到该卡点业务的具体名称和**绝对收入**、**同比增速**
3. 对比公司整体增速、行业平均增速
4. 查过去 3-4 个季度该业务的增速**趋势**（环比改善 / 持平 / 恶化）
5. 查估值（PE-TTM）与业绩增速的**匹配度**
6. 至少满足以下之一才进 Step 5 受益标的：
   - 业绩 +30% 以上 + PE 合理
   - 业绩 +10% 以上 + PE < 50x
   - 业绩持平 + PE < 30x（困境反转）

**「四件套」已不够，必须升级到「六件套」**。SKILL.md Step 5 已硬性要求第 5-6 件。
